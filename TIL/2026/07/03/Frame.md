 #Frame

 ## frame이란 무엇인가?
`frame` 안에는 프레임 이미지가 숫자 numpy 배열로 표현된다.

우선, 영상 크기가 만약 `1920 x 1080` 이면

```python
print(frame.shape)
```

```
(1080, 1920, 3)
```

로 출력된다.

의미는 각각 **(세로 픽셀 수, 가로 픽셀 수, 색상 채널 수)** 이다.
OpenCV에서는 색상 채널을 RGB가 아니라 BGR로 사용한다.

이미지 전체는 다음과 같이 생겼다고 보면 된다.

```
frame
=
[
  [픽셀, 픽셀, 픽셀, 픽셀, ...],
  [픽셀, 픽셀, 픽셀, 픽셀, ...],
  [픽셀, 픽셀, 픽셀, 픽셀, ...],
  ...
]
```

각 픽셀은 다시 색상값 3개를 가진다.

```
픽셀 = [B, R, G]
```
