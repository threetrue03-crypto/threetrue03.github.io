# \_\_doc__ 란?
이것은 파이썬 기본 문법인, \_\_doc__ 속성이라고 하며 클래스 \_\_doc__ 와 함수 \_\_doc__ 로 나뉜다.
클래스 선언, 함수 선언 바로 아래 저렇게 문자열만 적어 놓고 .\_\_doc__ 로 접근할 수 있다.

```python
class A:
    """
    A 클래스 설명입니다.
    """

    x = 10

print(A.__doc__)
```
```python
def hello():
    """
    인사하는 함수입니다.
    """
    print("hi")

print(hello.__doc__)
```
