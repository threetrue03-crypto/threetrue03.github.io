#json

## json이 무엇인가?
json은 파이썬 딕셔너리와 비숫한 구조를 가지는 텍스트 형식이다.

보통 데이터를 저장하거나 주고받을 때 많이 사용한다.

예를 들면 이런 형태를 가진다.

```json
{
  "frame_index": 10,
  "timestamp_sec": 5.0,
  "filename": "frame_00010.jpg"
}
```

파이썬에서는 보통 다음과 같은 데이터와 연결된다고 보면 된다.

```python
metadata = {
    "frame_index": 10,
    "timestamp_sec": 5.0,
    "filename": "frame_00010.jpg"
}
```

json은 이런 딕셔너리 같은 데이터를 파일로 저장하거나, 다른 프로그램과 주고 받기 좋은 텍스트 형태이다.

## python에서 어떻게 사용하는가?
만약 다음과 같은 python 데이터가 있다고 하자.

```python
metadata = [
    {
        "frame_index": 0,
        "timestamp_sec": 0.0,
        "filename": "frame_00000.jpg"
    },
    {
        "frame_index": 15,
        "timestamp_sec": 0.5,
        "filename": "frame_00015.jpg"
    }
]
```

### json.dump()란?
`json.dump()`란 파이썬 데이터를 json파일로 저장하고 싶을 때 사용한다.

```python
import json

with open("metadata.json", "w", encoding="utf-8") as f:
  json.dump(metadata, f, ensure_ascii=False, indent=2)
```

#### `ensure_ascii`는 json으로 저장할 때 모든 문자를 ascii 문자만으로 보장(ensure)할지 결정하는 옵션이다.

저 값이 False라는 건, 아스키 문자만으로 출력하는 것을 보장하지 않는다는 뜻이기 때문에 다른 문자를 그대로 저장한다.

하지만 만약에 `True`라면, 모든 문자를 아스키로 표현하여 저장한다.

다음 예시를 봐보자.

```python
data = {
    "message": "안녕하세요"
}
```

이런 데이터를 저장한다고 했을 떄 json에는 다음과 같이 저장된다.

```json
{
  "message": "\uc548\ub155\ud558\uc138\uc694"
}
```

하지만 False라면

```json
{
  "message": "안녕하세요"
}
```

로 저장된다.

#### `indent`는 들여쓰기 간격이다.
json파일을 보기 좋게 정리해주는 옵션이다.

만약 indent 없이 저장하면 json파일이 한 줄로 저장될 수 있다.

```json
{"video":"song.mp4","fps":30,"frames":[{"frame_index":0,"timestamp_sec":0.0},{"frame_index":15,"timestamp_sec":0.5}]}
```

이는 사람이 보기 매우 불편하기 때문에 indent값을 적어도 2로 설정해주는 것이 좋다.

### json.load()란?
Python에서 json을 읽어오는 함수이다.

아까 Python의 딕셔너리와 매우 비슷하다고 하였다. 다음과 같은 코드 예시도 딕셔너리를 이용한다.

```python
import json

with open("student.json", "r", encoding="utf-8") as f:
    student = json.load(f)

print(student)
print(student["name"])
print(student["scores"]["python"])
```

출력 결과는 다음과 같다.

```python
{'name': '조세진', 'student_id': '20252731', 'major': 'AI소프트웨어학부', 'grade': 1, 'scores': {'python': 95, 'math': 88, 'computer_vision': 92}}
조세진
95
```

### json.dumps()란?
json을 dump하여 파일로 저장하는 것이 아닌 `dump+string`하여 json의 형식을 문자열로 사용하는 것이다.

```python
import json

student = {
    "name": "조세진",
    "student_id": "20252731",
    "major": "AI소프트웨어학부"
}

json_text = json.dumps(student, ensure_ascii=False, indent=2)

print(json_text)
```

``` python
{
  "name": "조세진",
  "student_id": "20252731",
  "major": "AI소프트웨어학부"
}
```

여기서 json_text는 파일이 아니라 string이다.
