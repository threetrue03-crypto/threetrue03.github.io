# _Thread

## thread란?
일반적으로 프로그램은 직렬처럼 동작한다. 여기에 쓰레드를 적용하게 되면 병렬적으로 작업을 처리할 수 있게 된다.

전체적인 쓰레드 실행 흐름은 다음과 같다.

```
Python
├─ 스레드 1: 작업 A → 작업 B → 작업 C
└─ 스레드 2: 작업 X → 작업 Y → 작업 Z
```
